## 🎨 Über das Projekt

Professionelle Business-Website für die Kreativagentur **Maku-Hanga**, erstellt mit modernster Web-Technologie:

### 🚀 Technologie-Stack
- **React 19** - Moderne UI-Bibliothek
- **TypeScript** - Type-safe Entwicklung
- **Tailwind CSS v4** - Utility-first CSS Framework
- **Framer Motion** - Smooth Animations
- **Vite** - Blazing-fast Build Tool

### ✨ Design-Features
- **Dark Theme** mit Orange-Akzent (#ff6b35)
- **Grain Texture** Overlay für Premium-Look
- **Glassmorphism** Effekte
- **Responsive Design** - Mobile-first Approach
- **Smooth Animations** - Floating badges, shine effects
- **Accessibility** - WCAG konform

### 📱 Website-Sections
1. **Hero Section** - Animated gradient text & floating badge
2. **Services** - 4 interactive cards with hover effects
3. **Portfolio** - Gallery with scale-on-hover
4. **Team** - Member profiles with social links
5. **Contact** - Form with validation
6. **Footer** - Professional layout

## 🛠️ Lokale Entwicklung

```bash
# Repository klonen
git clone https://github.com/wore2706-lgtm/maku-hanga-pro.git
cd maku-hanga-pro

# Entwicklungsserver starten
npm install
npm run dev

# Build für Produktion
npm run build

# Vorschau des Builds
npm run preview
```

## 🎯 Design-System

### Farbpalette
- **Primary:** `#0a0a0a` (Near-black)
- **Secondary:** `#141414`
- **Accent:** `#ff6b35` (Orange)
- **Text Primary:** `#fafafa` (Warm white)
- **Text Secondary:** `#a1a1a1`

### Typografie
- **Headlines:** Clash Display (bold, distinctive)
- **Body:** Plus Jakarta Sans (refined readability)

### Animationen
- **Floating Badge:** 6s ease-in-out infinite
- **Shine Effect:** 12s linear infinite gradient
- **Hover Effects:** Scale, translate, border-color
